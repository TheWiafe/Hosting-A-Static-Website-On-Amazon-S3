My AWS re/Start Program Experience

As part of my new year's resolution for 2024, I consciously planned to advance my knowledge and skill set in the field of technology, specifically cloud computing and data analytics. Subsequently, I applied for the AWS re/Start program - a skill development program for entry-level careers in the cloud, offered by AmaliTech gGmbH in collaboration with Amazon Web Services (AWS). It was truly an enriching experience developing in-demand cloud based skills, while using cutting edge technology solutions. From working in the lab using Amazon Web Services (AWS)'s compute, storage, database, and networking software solutions to learning about the global AWS infrastructure (regions, availability zones and edge locations). I also had the opportunity of practically coding in Python, using MySQL (open source relational database management system), Linux, and Bash scripting. In addition to gaining these technical skills, I also had the opportunity to attend professional skill training sessions - which prepared all participants with communication, emotional intelligence, workplace collaboration, teamwork, resume writing, and job interviewing skills.
